### Get Generation History
```http
GET /api/generations?page=1&limit=10
```

**Query Parameters:**
- `page` - Page number (default: 1)
- `limit` - Items per page (default: 10, max: 50)
- `platform` - Filter by platform (optional)
- `tone` - Filter by tone (optional)

**Response:**
```json
{
  "success": true,
  "generations": [

  ],
  "pagination": {
    "currentPage": 1,

    "totalItems": 47
  }
}
```
